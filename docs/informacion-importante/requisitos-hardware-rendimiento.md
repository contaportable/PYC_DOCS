# Requisitos y Rendimiento

## Hardware
| Topico | Minimo | Recomendado |
| --- | --- | --- |
| Procesador | Core i5 1.5 ghz | Core i7 1.8 ghz |
| RAM | 8 GB | 16 GB |
| Disco | 200 MB | 500 MB |
| Sistema operativo | Windows 10 | Windows 11 |

## Conectividad
Se sugiere internet de 100 Mbps para volumen alto de descargas.
