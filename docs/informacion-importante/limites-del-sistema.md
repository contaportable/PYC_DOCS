# Limites del Sistema

## Alcance y limites
- No accede a bandejas ocultas.
- Puede fallar con servidores sin soporte IMAP.
- Actualmente no se cuenta con soporte para POP3.
- Puede verse limitado en cuentas Microsoft con permisos restringidos.
- Puede verse limitado en cuentas Google con control parental.
- Únicamente se puede agregar documentos fiscales fisicos a nivel de anexos, no a nivel de base de datos.
- Una mala conexion o falta de permisos puede entorpecer la lectura de correos, pero no afecta otras funciones de PyConta.
