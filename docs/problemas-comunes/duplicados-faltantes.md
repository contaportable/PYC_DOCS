# Duplicados o Faltantes

## Sintoma
Existen documentos repetidos o no se encuentra algun DTE esperado.

## Diagnostico rapido
1. Validar carpeta fuente y periodo.
2. Confirmar si hubo importacion manual adicional.
3. Revisar reglas de validacion activas.

## Solucion
1. Depurar registros duplicados.
2. Reimportar solo faltantes.
3. Ejecutar verificacion final por conteo.
