# DTE No Aparece

## Sintoma
El documento no se muestra en el listado después de descarga o importación.

## Diagnostico rapido
1. Confirmar rango de fechas activo.
2. Revisar filtros por tipo y contribuyente.
3. Verificar carpeta de origen.

## Procesos distintos
- Descarga: el DTE se obtiene desde el servidor o correo y se guarda en la carpeta local de PyConta.
- Importación: el DTE se carga desde un archivo local existente al sistema.

## Solucion
1. Ajustar filtros.
2. Si fue descarga, verificar que el archivo se haya guardado en la carpeta correcta.
3. Si fue importación, validar que el archivo seleccionado sea un DTE válido.
4. Si el problema persiste, puedes contactar con nuestro equipo de soporte usando [Soporte con PIN](../suscripcion-y-soporte/solicitar-soporte-con-pin.md) para asistencia personalizada.
