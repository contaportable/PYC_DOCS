# DTE No Aparece

## Sintoma
El documento no se muestra en el listado despues de descarga o importacion.

## Diagnostico rapido
1. Confirmar rango de fechas activo.
2. Revisar filtros por tipo y contribuyente.
3. Verificar carpeta de origen.

## Solucion
1. Ajustar filtros.
2. Ejecutar sincronizacion o importacion nuevamente.
3. Validar formato del archivo.
