# Configuracion Inicial

## Objetivo
Configurar correo y parametros minimos para iniciar la captura de DTE.

## Antes de empezar
- Correo empresarial disponible.
- Permisos IMAP habilitados cuando aplique.

## Pasos
1. Registrar cuenta de correo.
2. Seleccionar carpetas o buzones de lectura.
3. Guardar parametros de validacion base.
4. Ejecutar primera sincronizacion de prueba.
