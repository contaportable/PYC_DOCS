# Renovacion, Finalizacion y Upgrade

## Vigencia
El sistema se habilita por el tiempo contratado (6 o 12 meses segun plan).

## Finalizacion
Al vencer la suscripcion, se desactivan funciones operativas y se conserva acceso a informacion procesada.

## Renovacion
Se gestiona adquiriendo el mismo plan o uno diferente.

## Upgrade
Puede avanzarse a un plan superior; el calculo comercial aplica segun meses vigentes.
