# PyConta - Documentación Oficial

Esta documentación está diseñada para contadores y administradores que trabajan con alto volumen de DTE.

## Que es PyConta

PyConta es un software de escritorio desarrollado en Python para automatizar el procesamiento de Facturas Electronicas (DTE).

PyConta permite:

- Conectar Gmail, Outlook y otros servidores de correo para descargar automaticamente JSON y PDF.
- Validar y clasificar DTEs por contribuyente, periodo, categoria y tipo.
- Organizar una base de datos estructurada lista para uso contable y análisis. Facilitando la gestion de empresas, la visualizacion de DTEs, la generacion de anexos F07 y la exportacion a Excel.
- Mantener control local desde escritorio, sin dependencia permanente de la nube.
- Visualización gráfica de resumenes contables.

## Funciones clave

Las funciones de PyConta se concentran en tareas muy concretas:

- Creacion y configuracion de usuario local.
- Administracion y gestion de empresas.
- Descarga automatica de archivos JSON y PDF desde correo electronico.
- Selección de buzones.
- Sincronizacion de correo para lectura de carpeta Downloads
- Importacion manual de JSON desde carpeta local.
- Mostrar DTE en vista grafica, incluyendo lectura JSON y PDF.
- Generacion de CSV para anexos.
- Extraccion de datos a Excel.
- Categorizacion de DTEs por empresa, mes y categoria.
- Consulta del estado de DTE en Ministerio de Hacienda.
- Personalización de parámetros de validación y preferencias.
- Reglas de descarga para correos.
- Extraccion de DTEs para analisis.
- Generacion de archivo de metadata para procesamiento de IA.
- Autocompletado inteligente para columnas ISR del anexo F07.
- Predicciones globales para columnas ISR del anexo F07(compras).
- Anexos de Impuestos F07.
