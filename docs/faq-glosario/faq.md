# FAQ

## Preguntas frecuentes iniciales

### Que correos son compatibles?
Gmail, Outlook y otros proveedores compatibles con el metodo de acceso requerido.

### Que pasa si vence la suscripcion?
Se desactivan funciones operativas, pero se mantiene acceso a informacion procesada.

### El soporte es telefonico?
No. La atencion se brinda de forma remota por canales definidos.

### Se pueden procesar documentos fisicos?
No, el sistema no procesa documentos fiscales fisicos.
